2.What is GEC?

A.GEC stands for Global Execution Context. It is part of the JavaScript engine's execution process and plays a fundamental role in how JavaScript code is parsed and executed.


3.what is js engine?

A.A JavaScript engine is a program or interpreter that executes JavaScript code. It converts the human-readable JavaScript code into machine code that the computer can understand and run.


4.what is callstack?

A.The call stack is a data structure that tracks the function calls in a program, organizing them in the order they are invoked. It is used by the JavaScript engine to manage the execution of functions, maintaining the order in which code needs to be executed, including function calls and returns.
